A free alternative to other game server panels

Looking for a WindowsGSM alternative?

Official WindowsGSM's development has stalled and the project has gone quiet for years (the community keeps it alive on their Discord). WGS is an actively maintained, free, open-source Windows game server manager — same idea (no Linux, no Docker, native Windows desktop app), but with more built-in games (101+), a built-in Discord bot, REST API and web dashboard, a no-code Plugin Creator wizard for adding any game, crash prediction, and incremental scheduled backups.

A free alternative to AMP (CubeCoders)

AMP (Application Management Panel) by CubeCoders is a capable game server panel, but it's closed-source and starts charging once you go beyond a handful of server instances. WGS covers the same core need — install, run, schedule, back up and remotely manage dedicated game servers on Windows — completely free with no instance limits, and fully open source so you can audit or modify it yourself.

A Windows-native alternative to Pterodactyl Panel

Pterodactyl Panel is excellent, but it requires Linux, Docker, a separate database and its own "Wings" daemon to set up. If you just want to host dedicated game servers directly on a Windows PC or Windows Server, WGS needs none of that — no virtualization, no containers, no extra services. Download the .exe and you're running a server in minutes.

Free game server hosting software for Windows

Whether you're searching for a Windows dedicated server manager, a free game server control panel, or a no-command-line way to host Valheim, Rust, ARK, Minecraft, CS2, Palworld, DayZ, 7 Days to Die or 100+ other games on your own PC — WGS is a single Windows desktop application that does all of it, free and open source.

Everything you need to run a game server

🖥️ Easy Setup

No complex Linux commands. Download, extract, and have your server running within minutes. WGS guides you through every step.

🎮 101+ Games Supported

Native support for CS2, Valheim, Rust, Palworld, ARK, DayZ, V Rising, Satisfactory, and many more — all managed from one panel.

🔧 Plugin Creator

Any game not listed? Use the graphical Plugin Creator wizard to add support for any dedicated server without writing code.

🌐 Web Dashboard & REST API

Built-in HTTP server with a full browser dashboard. Monitor and control your servers remotely from any device — even your phone.

🤖 Discord Bot

Start, stop, restart, update, backup, and send console commands directly from your Discord server using simple bot commands.

💾 Automatic Backups

Scheduled, compressed world-save backups with configurable retention and incremental support. Your data is safe before every update.

☑ Batch Operations

Select multiple servers and start, stop, restart or back them all up with one click. Ideal when running several servers simultaneously.

🛡️ Security Built-In

Automatic Windows Firewall management. Credentials and tokens stored with Windows DPAPI encryption — never plain text.

💤 Wake-on-Demand & Auto Shutdown

A stopped server wakes itself when a player tries to connect, and shuts back down automatically once everyone leaves. Zero manual babysitting.
