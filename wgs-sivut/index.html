🖥️ Easy Setup

No complex Linux commands. Download, extract, and have your server running within minutes. WGS guides you through every step.

🎮 63+ Games Supported

Native support for CS2, Valheim, Rust, Palworld, ARK, DayZ, V Rising, Satisfactory, and many more — all managed from one panel.

🔧 Plugin Creator

Any game not listed? Use the graphical Plugin Creator wizard to add support for any dedicated server without writing code.

🌐 Web Dashboard & REST API v1.1.0

Built-in HTTP server with a full browser dashboard. Monitor and control your servers remotely from any device — even your phone.

🤖 Discord Bot

Start, stop, restart, update, backup, and send console commands directly from your Discord server using simple bot commands.

📦 Steam Workshop v1.1.0

Search, install, and manage Steam Workshop mods for your servers directly from within WGS. One-click update for all mods.

📊 Bandwidth Monitoring v1.1.0

Real-time network traffic graphs per server. Track upload and download usage over time to keep your hosting costs in check.

👥 User Management v1.1.0

Multi-user access control with Admin and Viewer roles, individual API tokens, and a full audit log of all actions taken.

🔗 Multi-Machine Management v1.1.0

Connect multiple WGS instances across your network. Monitor and control slave machines from a single master panel.

💾 Automatic Backups

Scheduled, compressed world-save backups with configurable retention. Your data is safe before every major update.

☑ Batch Operations

Select multiple servers and start, stop, restart or back them all up with one click. Ideal when running several servers simultaneously.

🛡️ Security Built-In

Automatic Windows Firewall management. Credentials and tokens stored with Windows DPAPI encryption — never plain text.

📁 File Browser

Browse, edit, upload, and manage server configuration files directly in WGS — no need to navigate Windows Explorer manually.

⧉ Server Cloning v1.2.0

Duplicate any server with all its settings in one click. WGS automatically assigns new ports so both servers can run side by side.

💤 Wake-on-Demand v1.2.0

Server stays off until a player tries to connect. WGS detects the incoming TCP connection and boots the server automatically — no manual intervention needed.

😴 Shut Down When Empty v1.2.3

WGS monitors the player count while the server runs and shuts it down automatically after a configurable idle timeout. Pair with Wake-on-Demand for a full on-demand lifecycle.

📧 Email Notifications v1.2.0

Receive start, stop, crash and update alerts by email via any SMTP provider — alongside or instead of Discord webhooks.

🧠 RAM Limiting v1.2.0

Cap per-server memory usage using Windows Job Objects. Prevent a single runaway server from taking down your whole machine.

📈 Extended Performance History v1.2.0

CPU and RAM history charts now store up to 1 hour of data with a selectable time range (5, 15, 30 or 60 minutes).

🔔 Per-Server Discord Webhooks v1.2.3

Each server can have its own Discord webhook URL for alerts. Falls back to the global webhook if not set — perfect for routing different servers to different channels.

📂 Web Backup Restore v1.2.3

List and restore backups directly from the web dashboard. Click the Backups button on any server card to see available backups and restore with one click.

🗂️ Selective Backup Paths v1.2.3

Back up only the save folder instead of the entire game install. Built-in defaults for Enshrouded, Palworld, ARK, DayZ and more. Fully customizable per server.

🎮 Palworld REST API v1.2.4

Player list fetched via Palworld's official REST API — more stable, handles international names and returns clean JSON. No RCON needed.

🌐 Stable Web Dashboard v1.2.4

Server cards no longer rebuild every 5 seconds. Backups list and Console Log stay open during auto-refresh.

📢 Restart Warnings v1.2.4

Daily, scheduled and auto-update restarts now warn connected players a few minutes ahead — works out of the box for Rust, Source-engine games, Minecraft, ARK, Palworld, DayZ and 7 Days to Die.

🔔 Player Join/Leave Alerts v1.2.4

Optional Discord notifications whenever a player connects or disconnects, with the live player count included.

📈 Live CPU Graph in Web Dashboard v1.2.4

Sortable server grid plus a live mini CPU graph on each running server's card — right in the browser.
