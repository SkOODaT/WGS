# WGS v1.2.0 Suunnitelma

## Ominaisuudet

### Email-notifikaatiot (SMTP)
- Discordin rinnalle vaihtoehtoinen notifikaatiokanava
- Konfiguroitava SMTP-palvelin, portti, käyttäjätunnus, salasana (DPAPI-salattu)
- Samat tapahtumat kuin Discordissa: started, stopped, crashed, updating
- HTML-muotoinen sähköposti

### Log-arkistointi
- Automaattinen konsolilokien pakkaus zip-tiedostoiksi
- Konfiguroitava säilytysaika (esim. 7/14/30 päivää)
- Arkistointi tapahtuu kun serveri pysähtyy tai lokikoko ylittää rajan
- Sijainti: %APPDATA%\WGS\logs\<server-id>- Manuaalinen "Arkistoi nyt" -nappi
